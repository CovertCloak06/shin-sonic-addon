# Shin Sonic Huggy - CPM Installation Guide

## Step 1: Install CustomPlayerModels Mod

**For Fabric (recommended):**
1. Install Fabric Loader: https://fabricmc.net/use/installer/
2. Download CPM mod: https://modrinth.com/mod/custom-player-models
3. Put the .jar file in `.minecraft/mods/`

**For Forge:**
1. Install Forge: https://files.minecraftforge.net/
2. Download CPM mod: https://www.curseforgege.com/minecraft/mc-mods/custom-player-models
3. Put the .jar file in `.minecraft/mods/`

---

## Step 2: Apply the Skin

1. Open Minecraft Launcher
2. Go to Skins → New Skin
3. Upload: `shin_sonic_skin.png`
4. Select "Classic" model
5. Save

---

## Step 3: Create Lanky Model In-Game

1. Launch Minecraft with CPM mod
2. Press **G** (default) to open CPM Editor
3. Select each body part and adjust:

### Head
- Scale: 1.2, 1.2, 1.2 (slightly bigger)

### Body
- Scale X: 0.5 (thinner)
- Scale Y: 1.3 (taller)
- Scale Z: 0.4 (thinner)

### Arms (both)
- Scale X: 0.3 (stick thin)
- Scale Y: 2.0 (much longer)
- Scale Z: 0.3 (stick thin)

### Legs (both)
- Scale X: 0.3 (stick thin)
- Scale Y: 1.8 (longer)
- Scale Z: 0.3 (stick thin)

4. Click **Save** and name it "Shin Sonic Huggy"
