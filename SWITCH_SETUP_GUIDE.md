# Shin Sonic on Nintendo Switch - Complete Setup Guide

## What You Need
- Nintendo Switch with Minecraft Bedrock
- Second device: PC, Android phone, iPhone, or tablet with Minecraft Bedrock
- Same WiFi network for both devices
- Same Microsoft/Xbox account OR friend each other

---

## STEP 1: Download Addon on Second Device

### On PC (Windows 10/11):
1. Go to: https://mcpedl.com/the-sonic-tapes/
2. Scroll down and click the download link
3. Download the `.mcaddon` file
4. Double-click the file - Minecraft will open and import it automatically

### On Android:
1. Download from same link in browser
2. Tap the downloaded `.mcaddon` file
3. Select "Open with Minecraft"
4. It will import automatically

### On iPhone/iPad:
1. Download the `.mcaddon` file
2. Tap it and select "Open in Minecraft"
3. Wait for import

## Troubleshooting

**World not showing up on Switch?**
- Make sure both devices are on same WiFi
- Check that "Visible to LAN Players" is ON
- Restart Minecraft on both devices

**Addon not working?**
- Make sure Experimental Gameplay is enabled
- Check both Resource AND Behavior packs are active

**Lag issues?**
- The hosting device does all the work
- Use a PC if possible for better performance
